# Hidden
Скрытый компонент ввода.

```jsx
return(
    <Form>
        <Hidden
            id='id'
            value={value}
        />
    </Form>
);
```

## Свойства и методы
|Prop name|Type|Default|Description|
|---------|----|-------|-----------|
|id|string||Идентификатор элемента ввода|
|value|string \| number||Значение элемента ввода|
